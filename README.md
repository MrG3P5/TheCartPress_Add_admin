### Install In TERMUX

```bash
> apt update && apt upgrade -y
> pkg install python python2 python3 git -y
> git clone https://github.com/MrG3P5/TheCartPress_Add_admin
> cd TheCartPress_Add_admin
> pip3 install -r requirements.txt
> python3 exploit.py
```
